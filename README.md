## Ausgewählte Projekte

### Device Inventory API

Produktionsnahes Spring-Boot-Backend zur Verwaltung von Netzwerkgeräten.

**Technologien:** Java 21, Spring Boot, PostgreSQL, Flyway, Docker, Spring Security, Swagger/OpenAPI, GitHub Actions

Highlights:

- REST API mit CRUD-Funktionen
- Rollenbasierte Absicherung
- PostgreSQL mit Flyway-Migrationen
- Docker Compose Setup
- CI/CD Pipeline
- Deployment auf VPS
- Swagger UI als Live Demo

Repository: https://github.com/pavel-sharipov/device-inventory-api

### IoT Fleet Telemetry Backend

Backend zur Verarbeitung und Speicherung von Telemetriedaten einer IoT-Flotte.

**Technologien:** Python, FastAPI, MongoDB, Docker, pytest

Highlights:

- REST API für Telemetriedaten
- asynchrone Verarbeitung
- Trennung zwischen Rohdaten und aktuellem Gerätezustand
- MongoDB Geo-Abfragen
- Dockerisierte Entwicklungsumgebung

Repository: https://github.com/pavel-sharipov/iot-fleet-backend
